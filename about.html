About TechRide.in

TechRide.in is a free news aggregating website which gets news from various popular sources that publish tech news from India.

It is developed and maintained by Vivekanand Padala, the founder of ManaLakshyam, a web and app development startup.

TechRide.in is built using HTML5, CSS3, Bootstrap 4.0, JavaScript, Jquery and ajax. The source code is available in GitHub.
